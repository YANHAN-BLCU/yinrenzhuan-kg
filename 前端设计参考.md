- 最大内容宽度：1100px（正文阅读舒适区）

- 卡片间距：`gap: 24px`
- 区块间距：`margin-bottom: 48px`

## 五、导航设计

### 5.1 顶部导航栏

- 固定在页面顶部，`backdrop-filter: blur(12px)` 毛玻璃效果
- 背景：`rgba(248, 244, 237, 0.85)`（半透明宣纸色）
- Logo 采用篆刻印章风格设计（朱砂红 + 墨色文字）
- 导航项hover时下划线从左向右滑入（`transition: transform 0.3s`）

```css
.nav-bar {
  position: fixed;
  top: 0;
  left: 0;
  right: 0;
  height: 64px;
  background: rgba(248, 244, 237, 0.85);
  backdrop-filter: blur(12px);
  border-bottom: 1px solid var(--border-light);
  z-index: 1000;
}
```

### 5.2 左侧分类边栏

- 按朝代（明代 / 清代）和流派（吴门印派 / 浙派 / 皖派 / 莆田派）组织
- 支持展开/折叠，高亮当前选中项
- 选中态：`background: rgba(193, 57, 45, 0.08); border-left: 3px solid var(--cinnabar);`

```css
.sidebar {
  width: 240px;

.sidebar-item {
```

## 十、参考网站详细说明

### 10.1 ctext.org（中国哲学书电子化计划）
**URL**: https://ctext.org/zh

古籍数字库的标杆。设计哲学是"内容为王"——无任何装饰性元素，以极高可读性为核心。左侧树形导航按"经·史·子·集"四部展开，检索框置于顶部。全程繁体支持、简繁切换、锚点跳转注释。对本项目的启发：传记类内容网站的导航结构与阅读体验设计。

### 10.2 识典古籍
**URL**: https://www.kvlab.org/project/shidianguji/

北京大学 × 字节跳动联合出品。暖白背景模拟宣纸质感，按经史子集分类。图文对照模式（数字影像与文本同屏）。异体字支持为平台特色。对本项目的启发：古籍原文与现代排版的平衡、繁简混排处理。

### 10.3 故宫博物院（dpm.org.cn）
**URL**: https://www.dpm.org.cn

博物馆网站的顶级参考。朱红+金色+墨黑的配色体系直接可提取使用。故宫文创专题展示了水墨动画加载效果、数字文物展示的交互模式。对本项目的启发：色彩体系的直接应用、印章意象的视觉表达。

### 10.4 花西子（huaxizi.com）
**URL**: https://www.huaxizi.com/

东方美学品牌的数字化呈现。"黛色"作为品牌主色，用现代设计演绎园林意境。品牌 Logo 采用印章风格设计。对本项目的启发：古典色彩的现代应用、品牌视觉体系的整体性。

### 10.5 非遗声景漫游馆
**URL**: https://www.cnblogs.com/yangsongduo/p/19413179

国潮风数字体验的代表性作品。使用褐色（榫卯）、米色（宣纸）、墨色（水墨）三色体系，模块化导航，墨滴扩散加载动画。对本项目的启发：传统色彩的工程化提取、CSS 动画实现。

### 10.6 思洋互动·国潮设计方法论
**URL**: https://www.ciya.cn/article/10153.html

提供了可直接使用的 CSS 色值变量定义、字体层级规范、动画实现代码。是对本设计文档技术细节最有价值的参考来源。

### 10.7 墨梅 (Momei) 博客主题
**URL**: https://github.com/CaoMeiYouRen/momei

水墨风博客主题的开源实现。提供了深浅双模式配色方案、排版优化参数（字号增大、行高充足）。可直接参考其 CSS 变量命名与排版规范。

### 10.8 台北故宫博物院（npm.gov.cn）
**URL**: https://www.npm.gov.cn/

以"宁静淡雅"为核心设计语言。色彩饱和度极低（暖白、淡青、淡赭），整体氛围营造出"在欣赏艺术品"的宁静感。对本项目的启发：意境美在 UI 中的表达方式——不靠堆砌元素，而靠克制与留白。
